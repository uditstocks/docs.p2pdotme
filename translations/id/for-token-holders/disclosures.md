Token $P2P adalah token utilitas dan tata kelola. Token ini bukan merupakan token yang mengacu pada aset atau token uang elektronik, dan bukan pula ekuitas, utang, atau sekuritas. Token ini membawa hak tata kelola dan utilitas protokol, tanpa hak penebusan dan tanpa klaim atas aset, pendapatan, atau keuntungan penerbit maupun yayasan. Nilai, jika ada, berasal dari penggunaan protokol dan partisipasi jaringan, bukan dari janji imbal hasil apa pun. Orang AS (US persons) dikecualikan dari penjualan perdana.
