## Token Tim (30% / 7,74 Juta)

Token tim mengikuti model paket kinerja MetaDAO. Alih-alih vesting berbasis waktu, token dibuka kuncinya berdasarkan tonggak kinerja protokol.

- 5 tranche yang sama, masing-masing sebesar 1.548.000 token
- Tranche dibuka kuncinya pada harga 2x, 4x, 8x, 16x, 32x dari harga ICO
- Cliff 12 bulan sejak peluncuran. Tidak ada tranche yang dibuka sebelum bulan ke-12.
- Setelah cliff, tranche hanya dibuka ketika ambang harga terpenuhi. Tidak ada jadwal kalender tetap di luar cliff.
- Harga diukur melalui TWAP 3 bulan setelah cliff

Karena setiap tranche dibatasi oleh ambang harga yang berkelanjutan, pembukaan kunci tim mencerminkan kinerja protokol, bukan waktu yang telah berlalu.

## Token Early Backer (20% / 5,16 Juta)

- Bulan 0 hingga 12 sepenuhnya terkunci (cliff)
- Pada bulan ke-12, 1.032.000 token dibuka kuncinya (20%)
- Pada bulan ke-15, 18, 21, dan 24, masing-masing 1.032.000 token dibuka kuncinya per kuartal
- Sepenuhnya ter-vest pada bulan ke-24

Vesting diberlakukan melalui kontrak vesting on-chain. Token yang terkunci tidak dapat di-stake.
