| Peran       | Gaji Bulanan                                    |
| ----------- | ----------------------------------------------- |
| "Keymaster" | Hingga $200                                     |
| "Cipher"    | Hingga $100 (berdasarkan tugas yang diberikan)  |

Monthly Spotlight memberikan hadiah $500 kepada masing-masing dari tiga penerima, sehingga total hadiah bulanan adalah $1.500. Semua pembayaran dilakukan langsung ke dompet P2P Anda.

| Item                                       | Jumlah | Tarif        | Total              |
| ------------------------------------------ | ------ | ------------ | ------------------ |
| Keymasters (4 negara x 4 jalur)            | 16     | Hingga $200  | $3.200 (maks)      |
| Ciphers (16 x 5)                           | 80     | Hingga $100  | $8.000 (maks)      |
| Monthly Spotlight                          | 3      | $500         | $1.500             |
| **Total pengeluaran bulanan**              |        |              | **$12.700 (maks)** |

**Alur persetujuan dana.** "Keymaster" mengajukan permintaan paling lambat tanggal 5 setiap bulan, "Country Lead" meninjau, "Core Team" memberikan persetujuan paling lambat tanggal 10, dan pencairan dilakukan paling lambat tanggal 15.
