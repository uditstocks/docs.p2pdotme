## COMPRA (Fiat para Stablecoin)

1. Crie uma ordem de `BUY`.
2. Receba os dados de pagamento do comerciante atribuído.
3. Envie o valor em fiat pelo canal instruído.
4. Conclua a confirmação exigida no aplicativo.
5. Acompanhe a ordem até a conclusão.

## VENDA / PAGAMENTO (Stablecoin para Fiat ou Canal de Pagamento)

1. Crie uma ordem de `SELL` ou `PAY`.
2. Aprove e transfira a stablecoin quando solicitado.
3. Aguarde a contraparte liquidar para o seu destino.
4. Confirme a conclusão no aplicativo assim que a contraparte tiver liquidado, e acompanhe a ordem até o estado `COMPLETED`.
