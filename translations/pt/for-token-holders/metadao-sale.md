O token $P2P é lançado por meio de um mecanismo de venda no estilo MetaDAO, projetado para uma distribuição justa e com a comunidade em primeiro lugar.

1. Os usuários comprometem USDC durante uma janela de compromisso de 4 dias
2. Os fundadores definem um limite discricionário sobre o total de captação aceito
3. Se houver supersubscrição, alocações e reembolsos são distribuídos pro-rata
4. 10M de tokens são distribuídos proporcionalmente entre todos os participantes no lançamento
5. Após a venda, o tesouro destina 20% do USDC captado e 2,9M de tokens para pools de liquidez
6. A autoridade de cunhagem é transferida para o tesouro governado pelo mercado

Os usuários existentes do protocolo recebem uma alocação preferencial na mesma avaliação dos demais participantes da venda, com base em seu XP on-chain.
