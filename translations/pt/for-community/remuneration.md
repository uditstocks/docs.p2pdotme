| Função      | Salário Mensal                          |
| ----------- | --------------------------------------- |
| "Keymaster" | Até $200                                |
| "Cipher"    | Até $100 (conforme a função atribuída)  |

O Destaque Mensal concede $500 a cada um dos três contemplados, totalizando um prêmio mensal de $1.500. Todos os pagamentos são feitos diretamente na sua carteira P2P.

| Item                                    | Quantidade | Taxa       | Total              |
| --------------------------------------- | ---------- | ---------- | ------------------ |
| Keymasters (4 países x 4 trilhas)       | 16         | Até $200   | $3.200 (máx.)      |
| Ciphers (16 x 5)                        | 80         | Até $100   | $8.000 (máx.)      |
| Destaque Mensal                         | 3          | $500       | $1.500             |
| **Gasto mensal total**                  |            |            | **$12.700 (máx.)** |

**Fluxo de aprovação de fundos.** O "Keymaster" submete a solicitação até o dia 5 do mês, o "Country Lead" revisa e o "Core Team" aprova até o dia 10, com desembolso até o dia 15.
